# InnovaCX — Frontend

React + Vite SPA for the InnovaCX customer experience platform.

---

## Stack

| | |
|---|---|
| **Framework** | React 19 |
| **Build tool** | Vite 7 |
| **Routing** | React Router v7 (lazy-loaded routes) |
| **Charts** | Recharts |
| **PDF** | @react-pdf/renderer |
| **HTTP** | Axios |
| **Styling** | Plain global CSS + CSS custom properties |

---

## Setup

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # dist/
npm run preview   # preview dist/
npm run lint      # ESLint
```

## Design System

CSS custom properties (defined in `src/index.css`):

```css
--gradient-brand      /* purple brand gradient */
--glass-bg            /* frosted glass background */
--btn-primary         /* primary button colour */
--radius-lg           /* large border radius */
--radius-pill         /* pill border radius */
--font-heading        /* display/heading font */
--text-dark           /* primary text colour */
--ease-out            /* standard easing */
--ease-spring         /* spring easing for popovers/toasts */
--dur-slow            /* slow animation duration */
```

**Page entrance:** `animation: fadeUp 0.4s var(--ease-out) both` on root containers.

**Public + Customer landing pages** use `pl-*` CSS class prefix (`PublicLanding.css`) to avoid collisions — the customer landing imports `PublicLanding.css` for the shared hero/features/footer structure.

**Toast pattern:** `{ show, message, type }` state + 4 s auto-dismiss + fixed bottom-right slide-up animation (`.ticket-toast` in `TicketDetails.css`).

**Inline modal error pattern:** red banner with ⚠ icon inside modal body, clears on input change (`.modal-inline-error` in `TicketDetails.css`).

## Auth

- `localStorage` stores `user`, `token`, and `access_token`
- `ProtectedRoute` wraps all authenticated routes and redirects to `/login` if no valid token
- Logout clears all four keys and navigates to `/login`

## Key Components

| Component | What it does |
|---|---|
| `useNovaChatbot` (`chatbot.js`) | Nova AI chatbot state + message handling |
| `CustomerFillForm` | Embeddable complaint submission form (used standalone + inside Nova widget) |
| `ProtectedRoute` | Role-aware auth guard |
| `KpiCard` | Reusable KPI metric card (manager/operator dashboards) |
| `ManagerTicketModalInner` | Controlled modal for reroute / rescore / escalate / resolve actions |
